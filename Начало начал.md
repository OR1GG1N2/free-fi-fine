## Запуск проекта:

Для запуска проекта выполните следующие команды:

```sh
npm i
npm run dev
```

В проект уже добавлен Auth.js, а переменные AUTH_DISCORD_ID и AUTH_DISCORD_SECRET находятся в файле .env.local.
